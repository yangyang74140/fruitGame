# 水果消除游戏 — 第一版开发规划

## 总览

- **目标：** 可试玩的完整 DEMO，包含 10 关 + 核心机制
- **引擎：** Cocos Creator 3.x + TypeScript
- **包管理：** pnpm

---

## 第一版开发阶段

### 阶段 0：项目初始化（1-2天）

- [x] GitHub 仓库创建
- [x] 项目文档（README / setup / demo方案）
- [x] 代码骨架（GameManager / LevelManager / RuleEngine / FruitItem / BasketSlot / UIManager）
- [x] 10 关关卡数据 JSON
- [x] 配置文件（fruit-config.json）
- [ ] Cocos Creator 项目创建（需本地 GUI）
- [ ] 场景搭建（主场景 + 预制体）
- [ ] 资源导入（6 种水果精灵图 + UI 素材）

**产出物：** 可在 Cocos Creator 中打开的空项目结构

---

### 阶段 1：基础循环跑通（2-3天）

**目标：** 点击水果 → 入篮 → 3 消 → 胜负判定

**任务清单：**
| # | 任务 | 文件 | 说明 |
|---|------|------|------|
| 1.1 | 搭建主场景 | scenes/Main.scene | 水果区域 + 篮筐 + UI 层 |
| 1.2 | 水果预制体 | prefabs/FruitItem.prefab | 精灵图 + 高亮环 + 冰冻层 |
| 1.3 | 篮筐槽预制体 | prefabs/BasketSlot.prefab | 藤编篮子底图 + 水果锚点 |
| 1.4 | 挂载 GameManager | Main.scene | 连接所有子模块 |
| 1.5 | 实现点击交互 | FruitItem.ts | 点击检测 + 事件传递 |
| 1.6 | 入篮动画 | FruitItem.ts → flyToBasket | 跳跃飞行效果 |
| 1.7 | 3 消逻辑 | GameManager.ts → checkMatch | 遍历篮筐找 ≥3 同类 |
| 1.8 | 消除动画 | BasketSlot.ts → playMatchAnimation | 弹起 → 缩小消失 |
| 1.9 | 胜负判定 | GameManager.ts | 调用 RuleEngine |
| 1.10 | 联调通关 | - | 第 1 关完整跑通 |

**产出物：** 第 1 关可完整游玩

---

### 阶段 2：关卡系统（1-2天）

**目标：** 支持 10 关加载、切换、进度

**任务清单：**
| # | 任务 | 说明 |
|---|------|------|
| 2.1 | LevelManager 资源加载 | 对接 Cocos resources.load |
| 2.2 | 关卡切换流程 | 过关 → 下一关，失败 → 重开 |
| 2.3 | 层级遮挡逻辑 | layer 判断 + 上层水果清除后下层变可点击 |
| 2.4 | 10 关数据校验 | 每关手动试玩，验证数据正确 |

**产出物：** 10 关全部可加载游玩

---

### 阶段 3：特殊机制（1-2天）

**目标：** 冰冻水果 + 扩容道具完整可用

**任务清单：**
| # | 任务 | 说明 |
|---|------|------|
| 3.1 | 冰冻视觉 | 半透明冰块覆盖 + 浅蓝高光 |
| 3.2 | 解冻逻辑 | 点击 2 次解冻 / 使用保鲜盒 |
| 3.3 | 保鲜盒道具 | 点击后自动解冻一个随机冰冻水果 |
| 3.4 | 扩容道具 | 篮筐临时 +1 格（最多 3 次） |
| 3.5 | 道具数量 UI | 显示剩余数量 + 用完后按钮灰掉 |

**产出物：** 所有机制可交互

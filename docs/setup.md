## 推荐本地环境

### 1. 安装 Cocos Creator

建议使用 Cocos Creator 3.x（建议与你最终目标平台保持一致）。

### 2. 打开项目

由于当前仓库还是空仓库，建议在本地执行：

1. clone 仓库
2. 使用 Cocos Creator 打开仓库目录
3. 新建 TypeScript 项目
4. 创建后将项目文件提交回仓库

### 3. 版本建议

- Node.js: 20+ 或当前 LTS
- pnpm: latest
- Cocos Creator: 3.x
